# Strymon Librarian


## Description

The Strymon Librarian is a tool for managing MIDI presets. <br>
It was written in C++ and uses the Qt Application and UI framework. <br>

<img src="http://www.strymon.net/downloads/spl/ss1.png" alt="Screenshot"/>
